Abstract

Crypto5GT is a small-footprint cryptographic C++ library crafted for security of network functions in 5G network.
It's a minimal set of cryptographic utilities designed to provide secure encryption, integrity protection, Key derivation function (KDF) and random number generation.
Crypto5GT provides utilities for complex random number generators, symmetric encryption algorithms supporting :-

  • AES (AES-CBC, AES-CTR, and AES-CMAC modes)
  • Snow-3G
  • ZUC
Crypto5GT is very lightweight and has no dependency on external libraries and licenses.

Purpose

Crypto5GT aims to provide single security API for NAS/AS messages protection which internally either wraps around Openssl function or implements algorithm completely.
Ciphering and integrity protection utilities for Non-access stratum (NAS) messages are designed to accept user input for underlying algorithm.
Supported cryptographic algorithms for NAS are designed as per 3GPP standards.
